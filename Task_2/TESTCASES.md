## 1. Получить объявление по id
Шаги:
    1. Создание объявления:
    - Отправить POST запрос с JSON-данными:
     
            {
                "name": "<string>",
                "price": "<integer>",
                "sellerId": "<integer>",
                "statistics": {}
            }
    
    - Ответ должен содержать id созданного объявления.
